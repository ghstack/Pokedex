An interactive Pokédex web application built with HTML, CSS, and JavaScript that consumes data from a public Pokémon API to display information about different Pokémon.

This project allows users to search for Pokémon by name or number, view their image and name, and navigate through them using previous and next buttons.

The main goal of this project is to practice API consumption, asynchronous JavaScript, and DOM manipulation.
